About Berkeley PR

BerkeleyPR (BPR) is a statistic tracking platform for competitive Super Smash Bros. Melee players.

BPR was conceived as the successor to the wildly popular GarPR (RIP) Melee Power Ranking web application. We use the same Microsoft TrueSkill ranking algorithm with Challonge tournament data and aim to crowdsource the collection of player data by self-identification.

Goals

  1. Crowd source the collection of tournament data into a single public repository
  2. Maintain a dynamic, empirical ranking of players per region
  3. Expose crowd sourced data through a robust API
