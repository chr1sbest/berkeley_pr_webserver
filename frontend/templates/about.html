About Berkeley PR

BPR was conceived as the successor to the wildly popular GarPR (RIP) Melee Power Ranking web application. Building off what GarPR did well (TrueSkill algorithm, responsive user interface, open source, etc.), we are aiming to build an application that fills the void that GarPR left and solves the problems that GarPR wasn't able to.

Goals

  1. Crowd source the collection of tournament data into a single public repository
  2. Maintain a dynamic, empirical ranking of players per region
  3. Expose crowd sourced data through a robust API
